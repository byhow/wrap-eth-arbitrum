# Wrap WETH On Arbitrum

## Instruction
1. Make sure this is on node >= 20.x and pnpm >= 9.x, or just npm >= 10
1. Run `cp .env.example .env` and fill in your own config
1. Run `pnpm install`
1. `pnpm dev`
